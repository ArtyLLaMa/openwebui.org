Installation

Get started in minutes

Prerequisites

  • Ollama running at http://localhost:11434 (v0.4.0+ for vision support)
  • Node.js (latest LTS version)
  • Yarn package manager
Quick Start 
git clone https://github.com/ArtyLLaMa/OllamaUI.git
cd OllamaUI
yarn install
yarn dev
